title: "OpenClaw Plugin"
description: "Connect OpenNote with OpenClaw"
---

## Install
```bash
opennote plugins install openclaw
```

## What it does
- Captures assistant output as jots
- Recalls relevant context from notebooks or jots
- Works locally with the OpenNote API

## Notes
- Requires OpenNote web running (`opennote start`)
- Uses local API at `http://localhost:4812/v1`
