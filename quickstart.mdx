This guide provides everything you need to connect **Semoia** to your AI Development Environments (IDEs) and Chatbots using the Model Context Protocol (MCP).

<Note>
  **MCP (Model Context Protocol)** is an open standard that enables AI assistants to connect with external data systems and tools.
</Note>

## Quick Setup (Recommended)

The fastest way to install is using our automated script. This script automatically detects installed IDEs on your machine and configures them for you.

<Steps>
  <Step title="Run Installation Command">
    Open your Terminal and run the following command (supports macOS/Linux):

    ```bash Terminal
    curl -fsSL https://semobeta.vercel.app/scripts/install-mcp.sh | bash
    ```
  </Step>
  
  <Step title="Authenticate Account">
    The command will automatically open your browser to log in to Semoia.
    
    1. Log in to your Semoia account.
    2. Copy the **8-character Authentication Code** displayed.
  </Step>

  <Step title="Complete Configuration">
    Return to your Terminal:
    1. Paste the authentication code and press Enter.
    2. Select the IDEs you want to configure (all detected IDEs are selected by default).
  </Step>

  <Step title="Restart IDE">
    <Check>
      Restart your IDE to apply the changes. Semoia MCP is now ready to use!
    </Check>
  </Step>
</Steps>

## Manual Configuration

If you prefer to configure manually or if the Quick Setup doesn't work for you, follow the detailed instructions below for each tool.

**General Configuration Info:**

- **Server URL:** `https://semobeta.vercel.app/api/mcp`
- **API Key:** Get it at [https://semobeta.vercel.app/api-keys](https://semobeta.vercel.app/api-keys).

<Tabs>
  <Tab title="Cursor">
    *Best support, requires version 1.0+*

    <Steps>
      <Step title="Open MCP Settings">
        Press `Cmd + Shift + P` (macOS) or `Ctrl + Shift + P` (Windows/Linux) and select **"MCP: Open Settings (JSON)"**.
      </Step>
      
      <Step title="Update Configuration">
        Add the following to your `mcp.json`:

        ```json mcp.json
        {
          "mcpServers": {
            "semoia": {
              "serverUrl": "https://semobeta.vercel.app/api/mcp",
              "headers": {
                "Authorization": "Bearer sk_semoia_..." 
              }
            }
          }
        }
        ```
        <Warning>Replace `sk_semoia_...` with your actual API Key.</Warning>
      </Step>
    </Steps>
  </Tab>

  <Tab title="VS Code">
    <Steps>
      <Step title="Open Configuration File">
        Open `mcp.json` at:
        - **macOS:** `~/Library/Application Support/Code/User/mcp.json`
        - **Windows:** `%APPDATA%\Code\User\mcp.json`
      </Step>
      
      <Step title="Update Configuration">
        ```json mcp.json
        {
          "mcpServers": {
            "semoia": {
              "serverUrl": "https://semobeta.vercel.app/api/mcp",
              "headers": { "Authorization": "Bearer sk_semoia_..." }
            }
          }
        }
        ```
      </Step>
    </Steps>
  </Tab>

  <Tab title="Windsurf">
    <Steps>
      <Step title="Open Configuration File">
        Open `mcp.json` at:
        - **macOS:** `~/.windsurf/mcp.json`
        - **Windows:** `%APPDATA%\Windsurf\mcp.json`
      </Step>
      
      <Step title="Update Configuration">
        ```json mcp.json
        {
          "mcpServers": {
            "semoia": {
              "serverUrl": "https://semobeta.vercel.app/api/mcp",
              "headers": { "Authorization": "Bearer sk_semoia_..." }
            }
          }
        }
        ```
      </Step>
    </Steps>
  </Tab>

  <Tab title="Claude Desktop">
    <Steps>
      <Step title="Update Configuration">
        Add the following to `claude_desktop_config.json`:

        ```json
        {
          "mcpServers": {
            "semoia": {
              "command": "npx",
              "args": ["-y", "@modelcontextprotocol/inspector", "https://semobeta.vercel.app/api/mcp"],
              "env": { "Authorization": "Bearer sk_semoia_..." }
            }
          }
        }
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>
