This guide shows you how to connect **Semoia** to your AI tools (like Cursor or Claude) using the Model Context Protocol (MCP).

<Note>
  **MCP** is a simple way for AI tools to talk to other apps and data.
</Note>

## Quick Setup (Recommended)

The fastest way to start is with our automated script. It will find your AI tools and set them up for you.

<Steps>
  <Step title="Run the Command">
    Open your Terminal and run this command:

    ```bash Terminal
    curl -fsSL https://semobeta.vercel.app/scripts/install-mcp.sh | bash
    ```
  </Step>
  
  <Step title="Log In">
    The command will open your browser.
    1. Log in to your Semoia account.
    2. Copy the **8-character code** you see.
  </Step>

  <Step title="Finish">
    Go back to your Terminal:
    1. Paste the code and press Enter.
    2. Select the tools you want to set up.
  </Step>

  <Step title="Restart">
    <Check>
      Restart your AI tool (like Cursor) to finish. You are ready to go!
    </Check>
  </Step>
</Steps>

## Manual Setup

If the script doesn't work, you can set it up manually.

**You will need:**
- **Server URL:** `https://semobeta.vercel.app/api/mcp`
- **API Key:** Get one at [trysemoia.com/api-keys](https://semobeta.vercel.app/api-keys).

<Tabs>
  <Tab title="Cursor">
    1. Press `Cmd + Shift + P` (Mac) or `Ctrl + Shift + P` (Windows).
    2. Search for **"MCP: Open Settings (JSON)"**.
    3. Add this to the file:

    ```json mcp.json
    {
      "mcpServers": {
        "semoia": {
          "serverUrl": "https://semobeta.vercel.app/api/mcp",
          "headers": {
            "Authorization": "Bearer YOUR_API_KEY" 
          }
        }
      }
    }
    ```
    *(Replace `YOUR_API_KEY` with your actual key)*
  </Tab>

  <Tab title="VS Code">
    Open your `mcp.json` file and add:

    ```json mcp.json
    {
      "mcpServers": {
        "semoia": {
          "serverUrl": "https://semobeta.vercel.app/api/mcp",
          "headers": { "Authorization": "Bearer YOUR_API_KEY" }
        }
      }
    }
    ```
  </Tab>

  <Tab title="Windsurf">
    Open your `mcp.json` file and add:

    ```json mcp.json
    {
      "mcpServers": {
        "semoia": {
          "serverUrl": "https://semobeta.vercel.app/api/mcp",
          "headers": { "Authorization": "Bearer YOUR_API_KEY" }
        }
      }
    }
    ```
  </Tab>

  <Tab title="Claude Desktop">
    Add this to your `claude_desktop_config.json`:

    ```json
    {
      "mcpServers": {
        "semoia": {
          "command": "npx",
          "args": ["-y", "@modelcontextprotocol/inspector", "https://semobeta.vercel.app/api/mcp"],
          "env": { "Authorization": "Bearer YOUR_API_KEY" }
        }
      }
    }
    ```
  </Tab>
</Tabs>
