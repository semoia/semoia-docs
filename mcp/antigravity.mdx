---
title: "Antigravity"
description: "Manual configuration for Antigravity"
---

Add this to your Antigravity MCP config file.

OpenNote runs locally. Ensure `opennote` is installed and available in your PATH.

```json
{
  "mcpServers": {
    "opennote": {
      "command": "opennote",
      "args": ["mcp", "run"]
    }
  }
}
```
