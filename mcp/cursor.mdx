description: "How to configure Manual Configuration for Cursor"
---

Go to: **Settings → Cursor Settings → MCP → Add new global MCP server**

In Cursor 1.0 and later, manually edit `~/.cursor/mcp.json`:
- **macOS**: `~/.cursor/mcp.json`
- **Windows**: `%APPDATA%\Cursor\mcp.json`
- **Linux**: `~/.config/cursor/mcp.json`

See [Cursor MCP docs](https://docs.cursor.com/context/model-context-protocol) for more info.


```json
{
  "mcpServers": {
    "semoia": {
      "url": "https://semobeta.vercel.app/api/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY",
        "Origin": "https://semobeta.vercel.app"
      }
    }
  }
}
```

---

## Giải Quyết Sự Cố (Troubleshooting)

- **Lỗi 403 (Forbidden):** Đảm bảo đã thêm header `Origin` chính xác là `https://semobeta.vercel.app`.
- **Lỗi 401 (Unauthorized):** Kiểm tra lại API Key thay cho `YOUR_API_KEY`.
- **Mất kết nối:** Đảm bảo URL kết thúc bằng `/api/mcp` và không có khoảng trắng.
