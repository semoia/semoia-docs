---
title: "BoltAI"
description: "Manual configuration for BoltAI"
---

Follow these steps to configure OpenNote MCP in BoltAI:

- Go to **Settings → MCP**
- Add a new server with the following details:
  - **Type**: HTTP
  - **URL**: `http://localhost:4812/api/mcp`
  - **Headers**: none

Ensure the OpenNote web app is running (`opennote start`).
