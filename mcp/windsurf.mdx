description: "How to configure Manual Configuration for Windsurf"
---

Open your Windsurf MCP configuration:
- **macOS**: `~/.windsurf/mcp.json`
- **Windows**: `%APPDATA%\Windsurf\mcp.json`
- **Linux**: `~/.config/windsurf/mcp.json`

See [Windsurf MCP docs](https://docs.windsurf.com/windsurf/cascade/mcp) for more info.


```json
"mcpServers": [
  {
    "name": "semoia",
    "url": "https://semobeta.vercel.app/api/mcp",
    "headers": {
      "Authorization": "Bearer YOUR_API_KEY",
      "Origin": "https://semobeta.vercel.app"
    }
  }
]
```

---

## Giải Quyết Sự Cố (Troubleshooting)

- **Lỗi 403 (Forbidden):** Đảm bảo đã thêm header `Origin` chính xác là `https://semobeta.vercel.app`.
- **Lỗi 401 (Unauthorized):** Kiểm tra lại API Key thay cho `YOUR_API_KEY`.
- **Mất kết nối:** Đảm bảo URL kết thúc bằng `/api/mcp` và không có khoảng trắng.
