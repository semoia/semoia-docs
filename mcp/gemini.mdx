---
title: "Gemini CLI"
description: "Manual configuration for Gemini CLI"
---

Add this to your `mcp_config.json`:

OpenNote runs locally. Ensure `opennote` is installed and available in your PATH.

```json
{
  "mcpServers": {
    "opennote": {
      "command": "opennote",
      "args": ["mcp", "run"]
    }
  }
}
```
