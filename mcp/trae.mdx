---
title: "Trae"
description: "Manual configuration for Trae"
---

Add this to your Trae configuration using the Continue Plugin format.

OpenNote runs locally. Ensure `opennote` is installed and available in your PATH.

```json
"mcpServers": [
  {
    "name": "opennote",
    "command": "opennote",
    "args": ["mcp", "run"]
  }
]
```
