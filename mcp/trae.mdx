---
title: "Trae"
description: "How to configure Manual Configuration for Trae"
---

Add this to your Trae configuration using the Continue Plugin format.

```json
"mcpServers": [
  {
    "name": "semoia",
    "url": "https://semobeta.vercel.app/api/mcp",
    "headers": {
      "Authorization": "Bearer YOUR_API_KEY",
      "Origin": "https://semobeta.vercel.app"
    }
  }
]
```
