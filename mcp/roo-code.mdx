---
title: "Roo Code"
description: "Manual configuration for Roo Code"
---

Add this to your Roo Code MCP configuration:

OpenNote runs locally. Ensure `opennote` is installed and available in your PATH.

```json
{
  "mcpServers": {
    "opennote": {
      "command": "opennote",
      "args": ["mcp", "run"]
    }
  }
}
```
