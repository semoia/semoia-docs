### Gemini CLI

Add to your `mcp_config.json`:




```json
{
  "mcpServers": {
    "semoia": {
      "serverUrl": "https://semobeta.vercel.app/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}
```
