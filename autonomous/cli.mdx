---

The easiest way to connect Semoia to your AI tools is using our CLI tool. It automatically detects and configures supported clients like Cursor, VS Code, and Claude.

## CLI Installation

Run the following command in your terminal to start the automatic configuration process:

<CodeGroup>

```bash npm
npx @semoia/install-mcp
```

```bash pnpm
pnpx @semoia/install-mcp
```

```bash bun
bunx @semoia/install-mcp
```

```bash yarn
yarn dlx @semoia/install-mcp
```

</CodeGroup>

## Supported Clients

The CLI currently supports automatic configuration for:
- Cursor
- VS Code
- Windsurf
- Claude Desktop & Claude Code
- Antigravity
- Trae / Zed / BoltAI / Gemini (via CLI detection)
