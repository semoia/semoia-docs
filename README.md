# OpenNote Docs

This repo contains the OpenNote documentation source (Mintlify).

## Local preview
Install the Mintlify CLI:

```bash
npm i -g mint
```

Run the docs locally:

```bash
mint dev
```

Preview at `http://localhost:3000`.

## Notes
- Docs live in `docs/`
- Navigation is in `docs.json`
- This repo does not require a deployed docs site
